正在学习git版本管理控制器！


（1）创建文件夹命令：

1.1  mkdir 文件夹名
1.2  cd 文件名
1.3  pwd 用于显示当前目录

（2）创建版本库（仓库）

2.2  git init  初始化   （ls -ah 可查看隐藏的.git文件夹）

（3）添加文件到仓库中

3.1  git add 文件名
3.2  git commit -m "描述"

（4）查看最近的提交日志

4.1  git log （最新版本后面有HEAD^来显示标识） git log --pretty=oneline  (简介显示提交的日志信息)

（5）回退版本

5.1  git reset --hard HEAD^  （在当前最新版本回退到上一个版本）回退多个版本是：HEAD~Number

（6）查看文本内容

6.1  cat 文件名
